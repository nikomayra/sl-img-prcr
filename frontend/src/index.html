Serverless Image Processoring Project

Serverless image processing project that uses Azure Functions and Blob Storage. User can upload a start, middle or end flagged image which is uploaded to the images container in blob storage. If there are at least one of each available in the container at that time a Gif is created using those images (randomly selected). The original images are deleted and the Gif is added to the gifs container. The gif gallery on this site are then updated, up to a maximum limit for the page (oldest removed for newest).

Mostly just a tech project, but there is some potential for some fun gifs to be created :)

Upload an image (size limit 1 mb). Must be 250x250 px. Supported filetypes [.jpeg, .png, .tiff, .bmp]
